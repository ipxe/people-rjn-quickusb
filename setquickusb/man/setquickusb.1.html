NAME

setquickusb - Configure QuickUSB port properties  

SYNOPSIS


 Configures the QuickUSB port direction for each bit.

 

DESCRIPTION

USAGE: setquickusb --OPTION [ ARG ] DEVICE

setquickusb reads or sets the port parameters for a QuickUSB module. The value to be set is any integer from 0-255, specified in either decimal or: hexadecimal form; when reading, setquickusb returns values in hexadecimal.

OPTIONS:
        --outputs  DEVICE
              gets the current output-mask for DEVICE, and prints it.
        --outputs=0x12  DEVICE
              sets the output-mask to 0x12


        --setting 0x03  DEVICE
              gets the current value of option register 0x03, and prints it.
        --setting 0x03=0x02  DEVICE
              sets the option register at address 0x03 to a value of 0x02.


        --default-outputs  DEVICE
        --default-outputs=0x34  DEVICE
        --default-levels DEVICE
        --default-levels=0x56 DEVICE
              these should set the default output mask and values for the device
              at power-on. Not yet implemented in the kernel driver.

EXAMPLE:
         setquickusb --outputs=0xf0 /dev/qu0ga
            Result:  Port A has bits 7-4 set as outputs, and 3-0 as inputs.

DEVICE NAMES: DEVICE is the relevant QuickUSB device and port. For example:         /dev/qu0ga First QUSB device, General Purpose Port A
        /dev/qu0gb First QUSB device, General Purpose Port B
        /dev/qu0gc First QUSB device, General Purpose Port C
        /dev/qu0gd First QUSB device, General Purpose Port D
        /dev/qu0ge First QUSB device, General Purpose Port E

        /dev/qu0hc First QUSB device, High Speed Port, Control
        /dev/qu0hd First QUSB device, High Speed Port, Data

Note 1: the high-speed port uses the same pins as G.P. ports B,D. Note 2: the 16-bit HSP (/dev/qu0hd) is little-endian. Byte B is read first. Note 3: the RS232 serial ports and I2C are not implemented in this driver.

WARNING:         Setting the output mask on a port configured for high-speed
        (either hc, or the corresponding gb,gd) will MESS IT UP.
        Don't do it!
